1 – Crie um programa que permita o usuário digitar um numero qualquer. Imprimir na tela a tabuada deste numero. (for)

2 - Elabore um programa que exiba na tela uma sequencia de números naturais começando em I, terminando em F. (while)

3 - Crie um programa que, utilizando a estrutura “Do While”, permita que o usuário digite um numero, este numero deve ser par, mostrar na tela o numero digitado. Não permitir que o usuário digite um numero que não seja par. (do/while)
